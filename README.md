# Bitcoin Price Prediction using LSTM

--- 

## Overview

This project implements a Long Short-Term Memory (LSTM) neural network model for Bitcoin price prediction using historical cryptocurrency market data. The system performs data preprocessing, sequence generation, model training, and forecasting to analyze Bitcoin price trends over time.

--- 

## Features

Historical Bitcoin price analysis
Data preprocessing and normalization
LSTM-based deep learning model
Time-series forecasting
Model training and evaluation
Prediction visualization using graphs
Technologies Used
Python
TensorFlow / Keras

NumPy
Matplotlib
Scikit-learn
Jupyter Notebook

## Workflow

Load and preprocess Bitcoin historical data
Normalize dataset values
Prepare sequential training data for LSTM
Build and train the LSTM model
Predict Bitcoin prices
Visualize actual vs predicted values

---

## Model Used

The project uses an LSTM (Long Short-Term Memory) neural network, which is highly effective for sequential and time-series data prediction tasks due to its ability to learn long-term dependencies.
